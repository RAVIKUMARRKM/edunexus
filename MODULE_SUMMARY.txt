API ENDPOINTS
================================================================================

Classes:
  GET    /api/classes                    List all classes
  POST   /api/classes                    Create new class
  GET    /api/classes/[id]               Get single class
  PUT    /api/classes/[id]               Update class
  DELETE /api/classes/[id]               Delete class

Sections:
  GET    /api/classes/[id]/sections      List sections for class
  POST   /api/classes/[id]/sections      Create section
  GET    /api/sections                   List all sections
  GET    /api/sections/[id]              Get single section
  PUT    /api/sections/[id]              Update section
  DELETE /api/sections/[id]              Delete section

Subjects:
  GET    /api/classes/[id]/subjects      List subjects for class
  POST   /api/classes/[id]/subjects      Create subject

Timetable:
  GET    /api/classes/[id]/timetable     Get timetable
  PUT    /api/classes/[id]/timetable     Update timetable

================================================================================
DATABASE SCHEMA (RELEVANT MODELS)
================================================================================

Class
  - id, name, numericValue
  - academicYearId, classTeacherId
  - roomNo, capacity
  - Relations: sections, subjects, students

Section
  - id, name, classId
  - roomNo, capacity
  - Relations: students, timetableSlots

Subject
  - id, name, code, classId
  - type, isOptional
  - Relations: assignments, timetableSlots

TimetableSlot
  - id, sectionId, subjectId, teacherId
  - dayOfWeek, startTime, endTime
  - roomNo

USAGE EXAMPLE
================================================================================

// Import API client
import { classesAPI } from '@/lib/api-client';

// List classes
const classes = await classesAPI.list();

// Create class
const newClass = await classesAPI.create({
  name: 'Class 10',
  numericValue: 10,
  academicYearId: 'year_123',
  capacity: 40
});

// Add section
const section = await classesAPI.sections.create(newClass.id, {
  name: 'A',
  capacity: 40
});

// Add subject
const subject = await classesAPI.subjects.create(newClass.id, {
  name: 'Mathematics',
  code: 'MATH101',
  type: 'THEORY'
});
